# P5 meta-UI — 인쇄된 화면 속 화면

**레이아웃 권한:** 선택 상자·커서·소셜 포스트 프레임·아카이브 탭을 실제 앱 화면이 아니라 인쇄물 위의 그래픽 장치로 배치한다. 배경 사진과 선택 영역의 색 대비로 화면 속 화면을 만든다.

**드롭인:** `a desaturated halftone photograph printed on matte stock, one region turning full color inside a blue selection rectangle with four corner handles, a large pixel cursor printed at the frame edge, layered archive tabs carrying short technical labels as page furniture, unmistakably a designed print poster rather than an app screenshot`

**마감 후보:** 바코드 또는 파일 탭 에디션 번호 중 1~2개.

**팔레트:** 모노크롬 베이스 + UI 액센트 1색, 필요하면 종이색 1색.

**카피 안전:** 코드·해시태그는 자유 작성 존, 핵심 헤드라인만 따옴표로 정확히 1회. `promo_text_effect: printed_meta_ui`. 끝 토큰 `AR 4:5` 또는 `AR 9:16`.
